# STM32-PWM-Generator
# STM32 PWM 代码生成器 (STM32 PWM Configurator) 🛠️

一个专为嵌入式开发者设计的在线工具，用于快速生成 STM32F103 标准库 PWM 初始化代码。

[🚀 **在线使用 / Live Demo**](https://yukino387.github.io/STM32-PWM-Generator/)

## ✨ 主要功能

* **自动通道匹配**：选择定时器和引脚，自动计算对应的通道号。
* **智能重映射**：自动识别引脚是否需要重映射，并生成对应的 `AFIO` 代码。
* **精准频率计算**：暴力算法搜索最佳 `PSC` 和 `ARR` 组合，将频率误差降至最低。
* **高级设置**：支持自定义系统时钟、输出极性及函数名前缀。
* **双语代码注释**：生成的 C 代码包含详细的中文注释，适合学习与工程使用。

## 🛠️ 使用方法

1.  打开 [在线工具](https://yukino387.github.io/STM32-PWM-Generator/)。
2.  在左侧面板选择 **定时器** 和 **目标引脚**。
3.  调整 **频率** 和 **占空比**。
4.  (可选) 点击 "高级设置" 修改系统时钟或函数名。
5.  点击右侧的 **复制** 或 **下载** 按钮，将代码添加到你的 Keil/IAR 工程中。
